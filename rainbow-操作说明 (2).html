1 · 准备与启动

1把 MIDI 键盘用 USB 线连到电脑(即插即用,不用装驱动)。
2用 Chrome 浏览器打开后台网页(苹果 Safari 不支持键盘连接)。
3点右上角 "启动引擎",浏览器弹窗问是否允许 MIDI 时点 "允许"。
4状态栏显示 "已连接 …" 和 "音频:已激活" 即就绪。拍一下贴了色标的键,应出声。
必须用 Chrome 或 Edge。连接 MIDI 键盘的功能(Web MIDI)目前只有这两个浏览器稳定支持,Safari 用不了。

2 · 七个颜色 = 七个音

颜色按彩虹顺序对应 C 大调七个音,颜色只对应音高、与高低八度无关——所有的 do 都是红色。

do C 红
re D 橙
mi E 黄
fa F 绿
sol G 青
la A 蓝
si B 紫

贴纸怎么贴,见配套的"贴标对照表"网页。

3 · 三种玩法

模式发声规则适合
🎯 训练模式只有"训练音"里启用的音会响,其余键静默低龄孩子,聚焦少量音高
🎹 自由模式所有琴键都响(黑键也响,但不计入统计)当普通钢琴随便玩
🎵 跟弹儿歌跟弹进行中所有键都响,系统逐音引导已能主动找键的孩子

训练/自由在右侧"发声模式"切换。跟弹从左上角"跟弹儿歌"面板启动,启动后自动接管发声,结束后回到原来的模式。

训练音怎么加

点"训练音"里的颜色卡可手动开关。数据达标时顶部会弹提示,建议加入下一个音(顺序 do→sol→mi→re→la→fa→si),你可采纳或暂缓。加音顺序、天数次数门槛都能在"进阶判断参数"里调。

4 · 跟弹儿歌详细操作

1在"跟弹儿歌"面板的下拉框选一首曲子(带 ⭐ 的是你自己导入的)。
2点 "开始跟弹"。屏幕出现一个大色块,显示当前要弹的音(唱名 + 简谱数字 + 颜色),同时语音报出这个音。
3孩子在琴上找到那个颜色的键弹下去。弹对了——进度条前进,自动报下一个音;弹错了——色块抖一下、琴照样响、不前进,连错两次会自动把当前音再报一遍。
4整首弹完,语音会夸一句,"今日弹完曲数"+1,并提示可换一首或再来一遍。
按钮作用
▶ 试听用钢琴音色自动播放整首示范曲,先听一遍再跟弹;再点一次停止。需先启动引擎
🔊 语音提示:开 / 关开关语音报音。关掉后只看颜色不听声,适合安静环境或想让孩子靠颜色自己找
🔁 再听一次把当前这个音重新报一遍
结束中途退出跟弹,回到普通模式
判定不分高低八度。只要音名对就算对——琴上有几个 do,弹哪个都通过。所以孩子不用记具体位置,认颜色就行。

用自己的声音报音(唱名录音)

右侧"唱名录音"面板可以把七个唱名录成你自己的声音,跟弹时优先播放你的录音(自带准确音高),比系统朗读效果好得多。

  1. 点某个音的【录】,浏览器询问麦克风权限时点"允许";
  2. 点下后立刻对准琴的音高唱这个音(0.5~1秒),2秒自动停,开头静音会自动剪掉;
  3. 点【听】试听,不满意点【删】重录。建议先在琴上弹一下这个音找准再录。

报音优先级:你的录音 > 网页目录下的 voice/1~7.mp3 音频文件 > 系统朗读。注意:录音功能需要在 https 网址下使用(本地双击打开的网页不行);录音存在本机浏览器,不随 JSON 备份走,换电脑需重录。

语音音色因电脑而异。系统朗读用的是电脑自带的朗读功能,Windows、Mac 读出来的声音不一样,是正常现象,不是故障。若完全没声音,见第 7 节。用自己的录音可完全绕开这个问题。

5 · 导入自己的歌曲

内置的十几首都是不受版权限制的公共旋律。想加别的曲子,用"导入歌曲 JSON"功能,自己写一个曲谱文件导进去。

第一步:把曲子翻成简谱数字

用 1 2 3 4 5 6 7 分别代表 do re mi fa sol la si,一个音一个数字,用空格隔开。目前只支持这七个基本音,不区分高低八度、不带节奏(时值长短系统不判断,只按顺序一个个走)。

例:小星星开头"一闪一闪亮晶晶"= 1 1 5 5 6 6 5

第二步:做一个 JSON 文件

用电脑记事本(Windows)或文本编辑(Mac)新建一个文本文件,按下面格式写,存成 .json 结尾。可以一次放多首:

[
  { "name": "小星星", "seq": "1 1 5 5 6 6 5 4 4 3 3 2 2 1" },
  { "name": "我家的歌", "seq": "3 3 3 5 1 2 3" }
]
  • name 是曲名(显示在下拉框,超过30字会截断)。
  • seq 是简谱数字序列,空格分隔,只认 1–7。
  • 方括号 [ ] 、大括号 { }、逗号、引号都要用英文半角符号,中文标点会导致读取失败。

第三步:导入

1点"跟弹儿歌"面板里的 "导入歌曲 JSON"。
2选中你做好的 .json 文件。
3提示"成功导入 N 首"后,下拉框里就能选到(带 ⭐ 标记)。
导入的歌会跟着数据一起备份。导出 JSON 备份时会一并保存,换电脑导入备份后自编曲目也在。
版权提醒:你自己导入什么曲子、是否有权使用,责任由你承担。内置曲目已规避版权风险,但大量现代创作儿歌仍在著作权保护期内,商用传播前请自行核实。

常见格式错误

现象原因
提示"读取失败"多半是用了中文标点(,""【】),或少了逗号/括号。用英文符号重写
导入成功但那首点开是空的seq 里没有 1–7 的有效数字,检查是不是写成了别的字符
某个音跳过了该音写了 0 或 8 以上,超范围会被忽略

6 · 数据与备份

数据只存在当前这台电脑的浏览器里,不上传云端、不跨设备同步。换电脑、换浏览器、清缓存都会丢。所以:固定用同一台电脑同一个浏览器,并每周点一次"导出 JSON 备份"存好。

换电脑时,在新电脑打开后台点"导入 JSON 恢复"选择备份文件即可接续,训练数据和自编曲目都会回来。

7 · 常见问题

Q:拍键没声音?
依次查:①点了"启动引擎"并允许 MIDI 了吗;②状态是否"已连接";③音箱开了吗、电脑音量够吗;④训练模式下拍的是不是启用音(没启用的键本来不响);⑤是不是用了 Chrome。
Q:跟弹时语音不出声?
先确认"语音提示"是开的。若还是没有,是电脑系统朗读功能的问题——Windows 检查是否装了中文语音包;实在不行可关掉语音,只用颜色引导照样能玩。
Q:跟弹判定太严/太松?
判定规则是"音名对即通过、不分八度",不可调。若孩子总弹错,建议先回训练模式把单个音练熟,或选前几首练习曲(音更少)。
Q:状态一直"未连接"?
重新插拔 USB;换个 USB 口;刷新网页再点一次"启动引擎";确认用 Chrome。
Q:导入的歌怎么删?
当前版本没有单独删除按钮。若要清理,只能"清空全部数据"后重新导入需要的(会连训练数据一起清空,清空前务必先导出备份)。
